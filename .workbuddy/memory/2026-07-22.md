## 白噪音哄睡小程序 - 功能实现

基于 uni-app + Vue 3 + TDesign + UnoCSS 项目骨架，实现完整白噪音哄睡小程序：

### 主题色
- #07c160（微信绿）

### 新增文件
- `src/static/sounds.json` — 声音数据配置（3 分类 18 音源，url 待填充）
- `src/types/sound.ts` — TS 类型定义
- `src/utils/format.ts` — formatBytes 等格式化工具
- `src/utils/audioCache.ts` — 音频缓存（小程序 saveFile + H5 localStorage，跨端兼容）
- `src/utils/audio.ts` — 基于 getBackgroundAudioManager 的循环播放封装
- `src/store/player.ts` — Pinia player store（当前音源/播放状态/定时器）
- `src/store/settings.ts` — Pinia settings store（夜间模式等偏好）
- `src/pages/index/index.vue` — 首页（播放区+分类 tab+卡片网格+底部控制）
- `src/pages/me/me.vue` — 我的页（帮助与支持+关于应用）
- `src/pages/about/about.vue` — 关于我们
- `src/pages/privacy/privacy.vue` — 隐私政策
- `src/pages/agreement/agreement.vue` — 用户协议
- `src/pages/contact/contact.vue` — 联系客服

### 修改文件
- `src/store/index.ts` — 导出 player + settings
- `src/tabbar/config.ts` — 切换到 NO_TABBAR 策略
- `pages.config.ts` — tabBar 空值兼容处理

### 关键功能
1. JSON 管理白噪音数据（3 分类：安抚初生/自然万象/精选混音，共 18 条）
2. 首次播放下载音频 → saveFile 缓存 → 下次直接读本地
3. 定时关闭（15/30/45/60/90 分钟 + 不限时）
4. 我的页：添加到桌面、分享好友、清理缓存（显示缓存大小）、版本信息
5. 后台音频播放（getBackgroundAudioManager）
6. 跨端兼容（MP-WEIXIN / H5 条件编译）

### 构建验证
- `pnpm dev:mp-weixin` 构建成功，dist/dev/mp-weixin/ 输出正常
- 6 个页面全部编译通过

- package.json import 在 Vite 中可能被 import-analysis 拦截，改用硬编码版本号
